# docker-ci-android-app
Image used in CI for building Android apps.

## Base image

Base on Ubuntu 14.04.

## What's inside

- Android command line tools 24.4.1
- Platform tools
- Build tools 23.0.3
- Android 4.3.1/4.4.2/4.4w.2/5.0.1/5.1.1/6.0
- Android support library
- Google repository
- Gradle 2.10
- git
- wget
- unzip
